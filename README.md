<h1 align="center">POSJOO</h1>

Simple implementation of Majoo Point of Sales

### Setup
1. Copy env.example to .env
2. Fill the config
3. Run `go get ./...`

### Run The App
`go run ./cmd/posjoo/main.go`

### Test The App
`go test -v ./...`

### Build The App
`go build -o ./bin/posjoo ./cmd/posjoo/...`
